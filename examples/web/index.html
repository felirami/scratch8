Get the real thing

# Desktop (full chiptune audio + .s8 export on E/S)
cargo run --release --features full

# Build your own carts
cargo run --example pong
cargo run --example snake

# Or just the core (no_std / embedded friendly)
cargo build --target thumbv7em-none-eabihf --no-default-features

.s8 cartridges & "compile once, run anywhere"

In the desktop runner press E or S to export the current game as a real .s8 file (magic header + name + sections for code/gfx/sfx/meta). Future web / embedded runtimes will load them directly or extract the inner WASM payload for true WASM carts.

The same binary that runs in this browser also cross-compiles to bare-metal microcontrollers. All the drawing, the PSG synthesis, the Cart logic — pure Rust, no external game engines.
