SCRATCH-8 Web (wasm32-unknown-unknown + Canvas)

Thin web runner for the portable core. Uses only the raw FFI exports (s8_ffi_*) from the lib compiled to wasm. No wasm-bindgen / bindgen-shims in v1 — maximum portability alignment.

Click canvas to focus. Arrows + Z/X (or mouse buttons for painter) • 1/2/3 keys also switch • E/S on desktop exports .s8

How to build & run this (first steps toward web + shareable carts)

# 1. Add the wasm target (once)
rustup target add wasm32-unknown-unknown

# 2. Build the core lib for wasm (no host crates)
cargo build --target wasm32-unknown-unknown --lib --no-default-features

# 3. Copy the artifact next to this HTML (debug is fine for now)
cp target/wasm32-unknown-unknown/debug/scratch8.wasm .

# 4. Serve (any static server; browser file:// may block fetch for wasm)
python -m http.server 8080
# Then open http://localhost:8080

# Later with audio feature or full web support you may use:
# cargo build --target wasm32-unknown-unknown --lib --features wasm

The .s8 cartridge export (press E/S in the desktop runner) produces real files today. Future: drop a .s8 (or its inner .wasm) here and the runner will load carts dynamically. This is the beginning of "compile once, run anywhere" via WASM carts + .s8 wrapper (see GOALS.md + src/carts/mod.rs + lib.rs FFI notes).
